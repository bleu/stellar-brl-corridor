# Partner Console

Reference partner surface for channel partners (accountants, tax advisors, wealth advisors) earning revenue share via on-chain `partner_transfer` events.

> **Status: T0 stub.** Lands in T2. React + Vite + TypeScript + Tailwind + shadcn/ui.

What it will surface:

- Partner onboarding (invite, e-signature, KYC reference).
- Live `partner_transfer` event feed for originated volume.
- Revenue-share payout history (auditable on chain).
